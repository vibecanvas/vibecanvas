# A4 - terminal: use PartySocket for resilient PTY connection

## Plan
- Compare the raw PTY websocket client with the existing PartySocket-backed ORPC websocket service and keep the terminal API shape small.
- Swap the terminal connection path to PartySocket so transient disconnects can recover more gracefully without rewriting the terminal session model.
- Preserve existing PTY-specific behavior such as cursor resume, manual close, and event forwarding while improving connection resilience.
