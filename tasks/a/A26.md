# A26 - remember canvas position for each canvas locally
[Overview](../BASED.md)

Persist each canvas viewport locally so reopening a canvas restores the last local pan and zoom without changing the shared document.

## Context
- `packages/canvas/src/components/Canvas.tsx`
- `packages/canvas/src/runtime.ts`
- `packages/canvas/src/services/camera/CameraService.ts`
- `packages/canvas/src/plugins/camera-control/CameraControl.plugin.ts`
- `packages/canvas/tests/plugins/camera/CameraControl.plugin.test.ts`

## Plan
### Runtime wiring
- Pass the stable canvas row id into runtime config.
- Keep persistence local to `camera-control`.

### Camera behavior
- Add one absolute viewport setter on `CameraService`.
- Restore saved viewport before `hooks.init`.
- Persist viewport on camera change.

### Tests
- Cover restore, per-canvas separation, invalid storage fallback, and persistence after pan/zoom interactions.
