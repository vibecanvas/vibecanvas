- Dependency: build after A5 and A6.
- Test file lives at `apps/cli/tests/cli/delete/cmd.delete.test.ts` (alongside the existing `move/` and `query/` CLI harness tests) rather than `packages/canvas/tests/cli/delete.test.ts`, to reuse the spawn-based harness in `apps/cli/tests/cli/harness.ts`.
